Why isn't my ZIP Code in your data?

ZIP Codes represent mail delivery routes, not areas. To represent the area that the route is in, ZIP Code Tabulation Areas (ZCTA) are used. Some ZIP Codes, like those for a single building, are bundled into their surrounding ZCTA.

Modified ZCTA (modzcta) is a geography used to analyze health data. Areas with small populations are combined. This allows for more stable rate calculations by increasing the underlying population of smaller areas.

If your ZIP Code isn't in the COVID-19 data, use this map or table to see what modzcta it is in.
